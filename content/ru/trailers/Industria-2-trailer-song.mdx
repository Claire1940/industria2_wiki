## Почему этот трек из трейлера сложно определить (и как решить проблему)

Сложность с **песней из трейлера Industria 2** связана с решениями по аудиодизайну, а не только с отсутствием кредитов. Монтажёры часто накладывают музыку под диалоги, чтобы сохранить повествовательный ритм. Для атмосферы это отлично, но для поисковых систем — сложно.

### Типичные препятствия

| Препятствие | Влияние на определение музыки | Практическое решение |
|---|---|---|
| **Диалоги поверх вокала** | Распознавание не даёт совпадений | Тестируйте участки с меньшим количеством речи |
| **Короткая зацикленная фраза** | Слишком обобщённо для поиска по тексту | Добавляйте ключевые слова, связанные с игрой |

| **Невыпущенный/лицензированный кусок** | Нет результата в публичных каталогах | Следите за официальными каналами на предмет отложенного раскрытия |

В этом случае повторяющаяся строка даёт подсказку, но не полную идентификацию. Поэтому ваша стратегия должна быть **сначала верификация**:

- Собирайте доказательства из самого трейлера  
- Отслеживайте ритм официальных публикаций  
- Подтверждайте авторство только когда оно названо в метаданных студии/издателя/платформы
